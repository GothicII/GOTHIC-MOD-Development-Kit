4. Script erstellen

Jetzt brauchen wir natürlich ein Skript, das gesteuert werden soll. Dazu verlassen wir den Spacer und öffnen UltraEdit bzw. einen ähnlichen ASCII-Editor.
Diese Zeilen müssen nun geschrieben werden:
----------------------------------------------
FUNC VOID EVT_MOD_01 ()
{
                Wld_InsertNpc  (SCAVENGER,"WP_01");
};
-------------------------------------------------------------------
Damit erstellen wir eine Skriptfunktion mit dem Namen EVT_MOD_01. Darin wird ein Monster an den Waypoint WP_01 eingefügt.
Die Datei wird unter /Scripts/Content/Story/Events abgelegt und EventMOD.d genannt. Man kann die Funktion aber auch in eine bestehende Datei integrieren.
